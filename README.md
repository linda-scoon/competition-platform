# Competition Platform

Repository foundation for the exercise competition platform MVP.

## Quick start

1. Copy environment values:
   ```bash
   cp .env.example .env
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```

## Baseline checks

```bash
npm run lint
npm run typecheck
npm run test
```

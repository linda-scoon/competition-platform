# Glossary

## Challenge
A competition object users can join.

## Run
One submitted completion attempt for a challenge.

## Creator
The user who owns a challenge. This is a challenge-scoped state, not a platform role.

## Participant
A user who joined a challenge.

## Competition Verifier
A user assigned to verify runs for a specific challenge.

## Site Admin
Site operations role.

## Super Admin
Highest security/override role.

## Soft Lock
Challenge state triggered when the first participant joins.

## Full Lock
Challenge state triggered when the submission window opens.

## Verifier Pool
List of users eligible to be assigned as competition verifiers.

## Fallback Participant Verification
Restricted post-close mode where participants may verify each other’s runs if no active verifiers remain and the vote passes.

## Media Library
User-owned image asset system for profile and challenge visuals, subject to admin moderation.
